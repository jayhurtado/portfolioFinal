Git & Github

With Git I learned that it's a version control system that is used to help keep track of updates made. The main steps that you will always use are add, commit, push, and pull. Then you use Github to host the commands you have done on Git. It is a Web-based Graphical User Interface for to make it more interactive for the user.

Linux

With Linux I learned that it is an open-source operating system that uses command line. We were able to SSH into the AFS NJIT server to use the Linux operating system. There we learned how to make/change directories, create files, look at the entire list of files and directories with commands such as mkir, cd, touch, ls.

HTML & CSS

With HTML & CSS I learned the basics of using both and how the coexist with one another. HTML has tags/elements such as html, body, p, div, img, and much more. Inside the tag you can have attributes such as class, id, src.

These attributes or tags are then used in the CSS file to change the styling of the webpage or the content where these attributes lie. The way it does this is by using the attribute or tags as the selector, then you choose the property and value such as 'color: blue;' and then the content of that selector changes to blue. CSS has many other features as well like font, width, height, padding, margin, float, etc.

Bootstrap

With Bootstrap I learned the basics as well. It is a framework that makes it a lot easier to create responsive web-pages. It has many different short cuts inline to what you would have to do a lot more work in CSS such as p-3, mb-3 which would stand for padding: 3px; margin-bottom: 3px; Bootstrap also comes with plenty of layouts and examples ready to use for any type of webpage you are looking for.
